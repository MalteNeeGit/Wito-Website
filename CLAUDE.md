# Wito Marketing Website

## Projekt
Landing Page für Wito - KI-Assistent für Zahnarztpraxen.
Ziel: B2B SaaS Look + nahbares Maskottchen kombinieren.
Zielgruppe: Zahnarztpraxen (B2B) und Patienten.

## Tech Stack
- Vanilla HTML/CSS/JS
- Keine Frameworks
- Google Fonts: DM Sans

## Brand
- Primär: #d9ed92 (hell grün)
- Mitte: #b5e48c (mittel grün)  
- Dunkel: #99d98c (dunkel grün)
- Hintergrund: #1a1a1a (dark mode)
- Schrift: #e0e0e0

## Stil
- B2B SaaS: clean, professionell, modern
- Nahbar durch Zahnmaskottchen "Wito"
- Mobile First
- Subtile Animationen

## Seitenstruktur
- index.html (Landing Page)
- style.css
- script.js
- assets/ (Logo, Maskottchen, Icons)

## Wichtige Regeln
- Keine externen Libraries außer Google Fonts
- Immer Mobile First denken
- Kommentare auf Deutsch
- Animationen subtil halten

## Design System (UI UX Pro Max)
- Pattern: Feature-Rich Showcase + Trust
- Style: Trust & Authority
- Font: DM Sans als Hauptschrif, Plus Jakarta Sans als Sekundärschrift
- Effekte: Badge hover, smooth stat reveal, metric pulse
- VERMEIDEN: Playful design, AI purple/pink gradients
- Responsive: 375px, 768px, 1024px, 1440px
- Icons: Nur SVG (Heroicons/Lucide) - keine Emojis


## Typografie
- Primär: DM Sans (Fließtext, Navigation, Buttons)
- Sekundär: Plus Jakarta Sans (Headlines, Hero)
